# 📧 Validax — Email Validation & Contact Utility

Validax is a lean, client-side solution for real-time email validation and seamless contact form handling. It leverages the [emailvalidation.io](https://emailvalidation.io/) API to validate format integrity and deliverability, and integrates [Web3Forms](https://web3forms.com/) for frictionless, spam-protected message submission—no backend required.

---

## 🚀 Core Capabilities

- ✅ **Email Format & Deliverability Check** via `emailvalidation.io` API  
- 🔄 **Asynchronous Workflow** with loading spinner and robust error handling   
- 🌐 **Serverless Contact Form** powered by Web3Forms  
- ⚡ **Zero Framework Footprint**: pure HTML5, CSS3, and ES6+  
- 🔒 **Spam Protection**: reCAPTCHA or honeypot implemented via Web3Forms

---

## 🛠️ Technology Stack

- **Markup & Styles**: HTML5, CSS3  
- **Logic**: Vanilla JavaScript (ES6+)  
- **Validation API**: [emailvalidation.io](https://emailvalidation.io/)  
- **Form Handling**: [Web3Forms](https://web3forms.com/)
